# FM Radio using Arduino and PL102BH-7P / RDA5807M module

Refered from https://github.com/lucsmall/Arduino-RDA5807M

## PL102BH-7P / RDA5807M module pin out
```
     ________
SDA |   ___  | ANT
SCK |  |   | | NC
NC  |  |___| | R Out
NC  |   ===  | L Out
VCC |________| GND
```
## PIN out for Digispark ATtiny85
```
 P0 - SDA 
 P1 - Inbuild LED
 P2 - SCL
 P3 - CH switch
 P4 - V + switch
 P5 - V - switch
```
